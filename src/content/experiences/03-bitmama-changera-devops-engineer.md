## Role overview

As a DevOps Engineer at Bitmama/Changera, I was responsible for designing, managing, and optimizing Kubernetes-based production infrastructure across multiple services and environments, ensuring high availability and cost efficiency.

## Key responsibilities

- Designed and managed Kubernetes-based production infrastructure on DigitalOcean using Terraform.
- Implemented CI/CD pipelines with GitHub Actions, Helm, and ArgoCD to automate deployments.
- Collaborated with developers to troubleshoot issues across application, infrastructure, and networking layers.
- Set up monitoring and alerting using Prometheus and Grafana to improve system visibility and reduce MTTR.
- Automated cluster scaling to handle traffic spikes and optimize resource usage.

## Selected achievements

- Maintained 99% uptime for production infrastructure while managing costs effectively.
- Transformed manual deployment processes into automated, reliable workflows.
- Improved system observability and reduced mean time to recovery through proactive monitoring.
