## Role overview

As a Customer Reliability Engineer at Cloudkite.io, I supported and optimized cloud-native applications across AWS and GCP, ensuring reliability, performance, and smooth deployments for diverse client environments.

## Key responsibilities

- Supported and optimized cloud-native applications running on AWS and GCP.
- Improved deployment workflows through CI/CD using Jenkins, GitLab CI, and Bitbucket Pipelines.
- Provided incident response, technical support, and postmortems for client infrastructures.
- Worked with Node.js and Python services deployed on Kubernetes to ensure reliability and performance.
- Performed Kubernetes cluster upgrades across AWS and GCP environments.

## Selected achievements

- Enabled smooth and frequent code releases through improved CI/CD workflows.
- Maintained SLA compliance and high client satisfaction through proactive support and incident management.
- Ensured application reliability under production workloads across multiple cloud platforms.
